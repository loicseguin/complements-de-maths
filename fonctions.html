Les fonctions de base¶

Fonctions polynômiales¶

Une fonction de la forme

\[f(x) = a_n x^n + a_{n-1} x^{n-1} + \ldots + a_2 x^2 + a_1 x + a_0\]

est dite polynômiale. Ce type de fonctions comprend les fonctions linéaires, \(f(x) = ax + b\) et les fonctions quadratiques \(f(x) = ax^2 + bx + c\).

Fonction linéaire¶

Pour une fonction linéaire \(g(u) = cu + d\), on appelle \(c\) la pente et \(d\) l’ordonnée à l’origine. Le graphe d’une fonction linéaire est une droite. La figure suivante illustre la fonction linéaire \(h(z) = -\frac{1}{2} z + 3\).

Fonction quadratique¶

Le graphe d’une fonction quadratique \(f(x) = ax^2 + bx + c\), où \(a \ne 0\) est une parabole. Si \(a > 0\), la parabole est ouverte vers le haut, sinon elle est ouverte vers le bas. La figure suivante illustre le graphe de la fonction \(f(x) = 2x^2 - 3x - 1\).
