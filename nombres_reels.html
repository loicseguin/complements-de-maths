Les nombres réels¶

Ensembles de nombres¶

Nombres naturels

\(\mathbb{N} = \{1, 2, 3, \ldots\}\)

Les nombres naturels sont ceux qu’on utilise pour compter. Noter que dans notre cours, l’ensemble des nombres naturels n’inclut pas le zéro.

Nombres entiers

\(\mathbb{Z} = \{\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \}\)

Les nombres entiers comprennent tous les nombres naturels, le zéro et tout les nombres négatifs.

Nombres rationnels

\(\mathbb{Q} = \left\{\frac{a}{b} \left|\right. \, a \in \mathbb{Z}, b \in \mathbb{Z} \setminus \{ 0 \} \right\}\)

Les nombres rationnels inclut les nombres entiers et toutes les fractions. Il est important de noter qu’une fraction ne peut avoir un dénominateur nul.

Nombres réels

\(\mathbb{R} = \left\{a,b_1 b_2 b_3 \ldots | \, a \in \mathbb{Z}, b_i \in \mathbb{N} \cup \{0\} \right\}\)

Dans notre cours, les nombres réels sont définis comme les nombres qui admettent un développement décimal. Les nombres réels inclut les nombres rationnels et les nombres dit irrationnels, c’est-à-dire les nombres dont le développement décimal est infini et apériodique (par exemple, \(\pi\) et \(\sqrt{2}\)).

Opérations sur les nombres réels¶

Addition

L’addition de nombres réels possède cinq propriétés fondamentales. Pour tout \(a, b, c \in \mathbb{R}\),

  1. Fermeture: \(a + b \in \mathbb{R}\)
  2. Associativité: \((a + b) + c = a + (b + c)\)
  3. Commutativité: \(a + b = b + a\)
  4. Élément neutre: \(a + 0 = 0 + a = a\)
  5. Élément opposé: \(a + (-a) = (-a) + a = 0\)
Soustraction
La soustraction de deux nombres réels est définie comme l’addition de l’élément opposé: \(a - b = a + (-b)\). Ceci est équivalent à dire que \(a - b = c\) si et seulement si \(a = b + c\).
Multiplication

La multiplication de nombres réels possède cinq propriétés fondamentales. Pour tout \(a, b, c \in \mathbb{R}\),

  1. Fermeture: \(a \cdot b \in \mathbb{R}\)
  2. Associativité: \((a \cdot b) \cdot c = a \cdot (b \cdot c)\)
  3. Commutativité: \(a \cdot b = b \cdot a\)
  4. Élément neutre: \(a \cdot 1 = 1 \cdot a = a\)
  5. Élément inverse: \(a \cdot \frac{1}{a} = \frac{1}{a} \cdot a = 1\)

On note que la multiplication est une façon compacte de répéter une addition. Par exemple, plutôt que d’écrire \(2 + 2 + 2 + 2\), on utilise la multiplication \(2 \cdot 4\).

Division
La division est définie comme la multiplication par l’inverse: \(a \div b = a \cdot \frac{1}{b}\). Ceci est équivalent à dire que \(\frac{a}{b} = c\) si et seulement si \(a = b \cdot c\).

Au dix propriétés ci-dessus s’ajoute une propriété qui concerne la combinaison de l’addition avec la multiplication : la distributivité.

\[a \cdot (b + c) = a \cdot b + a \cdot c\]

Relation d’ordre¶

Il existe une relation d’ordre entre les nombres réels, c’est-à-dire qu’il est possible de classer les nombres réels dans un ordre précis. Pour toute paire \(x\), \(y\) de nombres réels, une, et une seule des trois relations suivantes est vraie :

  • \(x < y\) (\(x\) est plus petit que \(y\))
  • \(x > y\) (\(x\) est plus grand que \(y\))
  • \(x = y\) (\(x\) est égal à \(y\))

Valeur absolue¶

La valeur absolue d’un nombre réel \(x\) est définie comme

\[\begin{split}|x| = \begin{cases} x & \text{ si } x \geq 0 \\ -x & \text{ si } x < 0 \end{cases}\end{split}\]

Par exemple, \(|5,349| = 5,349\) car \(5.349\) est plus grand que zéro. On a aussi \(|-3\pi| = -(-3\pi) = 3\pi\) car \(-3\pi\) est inférieur à zéro.

Distance¶

La distance entre deux nombres réels \(a\) et \(b\) est la longueur du segment de droite reliant ces deux nombres sur la droite réelle. Si \(a \geq b\) alors \(d(a, b) = a - b\), si \(a < b\) alors \(d(a, b) = b - a\). On peut écrire cette définition de façon plus compacte en utilisant la valeur absolue :

\[d(a, b) = | a - b |\]

Il est important de se rappeler que la distance entre deux nombres est, par définition, positive ou nulle.

Puissances¶

On a vu que la multiplication correspond, pour les nombres entiers, à une addition répétée. De façon similaire, les puissances sont définies comme des multiplications répétées. Par exemple, plutôt que d’écrire \(8\cdot 8 \cdot 8 \cdot 8\), on peut écrire \(8^4\).

Ainsi, si \(n\) est un nombre naturel, on définit, pour tout \(x \in \mathbb{R}\), la \(n\)-ième puissance de \(x\) par

\[x^n = x \cdot x \cdot \ldots \cdot x\]

où \(x\) est répété \(n\) fois. À partir de cette définition, on peut démontrer les propriétés suivantes :

Pour tout \(x \in \mathbb{R}\), \(n, m \in \mathbb{N}\)

  1. \(x^n \cdot x^m = x^{n+m}\)
  2. \(\left( x^n \right)^m = x^{nm}\)

Pour définir les puissances pour tous les exposants entiers, on fait l’observation suivante. Pour obtenir \(x^2\) à partir de \(x^1\), il suffit de multiplier \(x^1\) par \(x\). De même, pour obtenir \(x^3\) à partir de \(x^2\), il suffit de multiplier par \(x\). En continuant ainsi, on définit toutes les puissances naturelles. Le processus fonctionne aussi à l’inverse. Par exemple, pour obtenir \(x^2\) à partir de \(x^3\), il suffit de diviser \(x^3\) par \(x\). Pour obtenir \(x^1\) à partir de \(x^2\), il suffit de diviser \(x^2\) par \(x\). En continuant ainsi, on peut définir \(x^0\) comme \(x^1\) divisé par \(x\), i.e.: \(x^0 = 1\). Ceci fonctionne pour tout \(x \in \mathbb{R}\setminus\{0\}\) car on ne peut pas diviser par zéro.

De même, \(x^{-1} = x^0 / x = 1/x\). Donc \(x\) à la puissance \(-1\) est l’inverse multiplicatif de \(x\). En général, \(x^{-n} = 1/x^{n}\). Les deux propriétés ci-dessus sont encore valides pour les exposants entiers.

Pour définir les exposant rationnels, on suppose que les deux propriétés ci-dessus sont valides et on définit \(x^{1/n}\) comme le nombre réel (positif, si \(n\) est pair) qui, lorsqu’on l’élève à la \(n\)-ième puissance, donne \(x\). Par exemple, \(8^{1/3}\) est le nombre réel qui, lorsqu’on l’élève à la troisième puissance donne \(8\). Évidemment, ce nombre est \(8^{1/3} = 2\). Si \(x\) est négatif, \(x^{1/n}\) n’est défini que pour \(n\) impair.

De façon générale, \(x^{a/b} = \left( x^{1/b} \right)^a\).

Il est fréquent de faire référence à \(x^{1/n}\) comme la n-ième racine principale de \(x\), notée \(\sqrt[n]{x}\). L’exemple le plus commun est la racine carrée qui correspond à la deuxième racine principale. On note que par définition, la racine carrée d’un nombre est toujours positive.

Avertissement

Soit \(x \in \mathbb{R}\). Alors \(\sqrt{x^2} = | x|\). Pourquoi ? Il suffit de considérer le cas où \(x\) est négatif. Par exemple, si \(x = -2\), alors \(x^2 = 4\) et \(\sqrt{x^2} = \sqrt{4} = 2 \ne x\).

Définir rigoureusement les puissances réelles en général demande beaucoup plus de travail. Ici, on se contentera de supposer qu’il est possible d’élever un nombre positif à n’importe quelle puissance réelle. Par exemple, \(5^\pi \approx 156,99\).
